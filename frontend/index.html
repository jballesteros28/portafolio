Proyectos

Gestor de tareas personales (To-Do List):

Desarrollo de una aplicación web moderna y segura para la gestión de tareas personales.

Permite registro de usuarios con confirmación por email, autenticación segura (JWT), recuperación de contraseña y administración completa de tareas (CRUD), todo en un entorno privado por usuario. Funcionalidades avanzadas: eliminación y recuperación de cuentas, limpieza automática de usuarios inactivos, mensajes de feedback y diseño neuromórfico, totalmente responsive. Stack: React (Vite), CSS clásico (diseño neuromórfico), FastAPI, SQLAlchemy, Alembic, PostgreSQL (Render), SMTP, JWT, Vercel (frontend), Git/GitHub.
